# Tracker Application
<h2>Description</h2>
A mobile application with backend that records user location over time and plots the path taken. Location data is stored and can be accessed by logging 
into your account

<br />


<h2>Languages and Utilities Used</h2>

- <b>React Native</b> 
- <b>JavaScript</b> 
- <b>Express</b>
- <b>Node</b>
- <b>MongoDB</b>
- <b>JWT</b>
